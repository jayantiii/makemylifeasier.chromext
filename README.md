# Highlight to Enlight

**Work smart, not harder. Just highlight and let AI do the rest.**

A Chrome extension that brings AI writing assistance directly to any webpage. Simply highlight text and get instant AI-powered responses without switching tabs or managing API keys.

---

## Features

- **Highlight Anywhere**: Select text from any webpage
- **Fully Customizable**: Change the system prompt to personalize AI responses with your profile and writing style
- **Instant AI Responses**: Get AI-powered writing assistance using Chrome's built-in AI
- **Zero Setup**: No API keys, no external dependencies
- **Works Offline**: Powered by Chrome's native AI

---

## Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/jayantiii/makemylifeasier.chromext.git
cd makemylifeasier.chromext
```

### 2. Enable Chrome Built-in AI

1. Go to `chrome://flags/`
2. Search for `writer-api-for-gemini-nano`
3. Set to **"Enabled"**
4. **Restart Chrome**

### 3. Load the Extension

1. Go to `chrome://extensions/`
2. Enable **"Developer mode"**
3. Click **"Load unpacked"** and select the extension folder

### 4. Customize Your System Prompt

1. Click the extension icon → **"Settings"**
2. Enter your profile, writing style, and preferences
3. Click **"Save"**

You can update your system prompt anytime to personalize all AI responses.

---

## Usage

1. **Highlight text** on any webpage
2. **Click the extension icon**
3. **Optionally add a custom prompt** (e.g., "Write a cover letter", "Fix grammar")
4. **Click "SEND TO LLM"**
5. **Copy the response**

---

## Examples

- **Cover Letters**: Highlight job description → "Write a cover letter for this position"
- **Email Fix**: Highlight draft → "Fix grammar and make this more professional"
- **Summarize**: Highlight text → "Summarize this"
- **Improve**: Highlight text → "Improve clarity and tone"

All responses are personalized based on your system prompt settings.

## Requirements

- Chrome Browser 138+
- Chrome flag: `writer-api-for-gemini-nano` enabled

---

## Troubleshooting

**Chrome AI not available?**
- Verify flags are enabled and Chrome was restarted
- Check Chrome version (needs 138+)

**Text selection not working?**
- Try the "Refresh Selection" button
- Refresh the webpage

**No system prompt set?**
- Click Settings → Enter your prompt → Save
